## Extractor2md

**Author:** neyara11
**Version:** 0.1.2
**Type:** tool

### Description

Dify plugin that extracts text and document structure from files and converts them to Markdown format. Detects headings from PDF (font analysis), Word (heading styles), HTML (`<h1>`-`<h6>`), and PowerPoint (slide titles). Outputs structured documents split by section headings for downstream RAG processing.

**Supported formats:** PDF, DOCX, XLSX/XLS, PPTX, HTML, Markdown, CSV, JSON, YAML, plain text.
